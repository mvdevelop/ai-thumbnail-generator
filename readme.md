
## 🎨 AI Thumbnail Generator
Crie thumbnails chamativas para vídeos e redes sociais em segundos utilizando o poder da Inteligência Artificial. Este projeto transforma descrições de texto em artes visuais otimizadas.

## ✨ Funcionalidades
🤖 Geração por IA: Integração com APIs de geração de imagem (DALL-E/Stability).
🖼️ Edição de Texto: Adicione títulos e sobreposições diretamente na imagem gerada.
📐 Formatos Predefinidos: Tamanhos otimizados para YouTube, Instagram e TikTok.
📥 Exportação Direta: Download em alta resolução (PNG/JPG).
🌓 Modo Escuro/Claro: Interface adaptável e moderna.

## 🚀 Tecnologias Utilizadas
React.js (Vite)
Tailwind CSS para estilização.
Lucide React para ícones.
Axios para requisições de API.
Canvas API / HTML2Canvas para renderização da thumb final.
